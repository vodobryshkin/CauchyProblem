# ЛР6 — фронтенд для задачи Коши

## Запуск

Backend должен быть запущен на `http://localhost:8080`.

```bash
cd front
npm install
npm run dev
```

Фронтенд откроется на `http://localhost:5173`.

## Что есть

- выбор одного ОДУ;
- выбор одного метода за один расчёт;
- ввод `x0`, `y0`, `xn`, `h`, `epsilon`;
- отображение параметров JSON-ответа списком;
- таблица только для массивов из JSON: `table.x_row`, `table.y_row`, `exact_solution`;
- график точного решения, вычисленного на фронтенде по формулам из `MathConfig`;
- график приближённого решения и отдельные точки узлов;
- KaTeX для отображения формул.
